Module Slacko

module Slacko: sig .. end
An OCaml binding to the REST API of Slack. Each function triggers an HTTP request, so it returns immediately and returns an Lwt.t value.
Author(s): Marek Kubica

Types used in the binding


type api_result = [ `Success of Yojson.Basic.json | `Unknown_error ]
The basic type that each API endpoint will always return. Most API calls return much more possible error cases, see below.
type auth_error = [ `Account_inactive | `Invalid_auth | `Not_authed ]
API calls that require authentication might fail with these errors
type timestamp_error = [ `Invalid_ts_latest | `Invalid_ts_oldest ]
API calls that take timestamps can throw errors when the timestamp is invalid.
type channel_error = [ `Channel_not_found ]
API calls that require Slacko.channel inputs can throw this error.
type user_error = [ `User_not_found ]
API calls that require Slacko.user inputs often throw this error if the user was not found.
type invite_error = [ `Cant_invite | `Cant_invite_self ]
Inviting might fail because invitation is impossible for some reason or because attempting to invite oneself.
type not_in_channel_error = [ `Not_in_channel ]
Some API calls require the Slacko.user to be in Slacko.channel for the action to succeed, not meeting this requirement can raise this error.
type already_in_channel_error = [ `Already_in_channel ]
Some API calls require the Slacko.user not to be in Slacko.channel for the action to suceed. The opposite of Slacko.not_in_channel_error.
type archive_error = [ `Is_archived ]
Channels might be archived, so modification attempts will fail with this error
type name_error = [ `Name_taken ]
When creating channels, names have to be unique, an attempt to create a duplicate one will result in this error.
type kick_error = [ `Cant_kick_self ]
Kick (in general) might fail, because kicking oneself is not supported.
type channel_kick_error = [ `Cant_kick_from_general | `Cant_kick_from_last_channel | `Cant_kick_self ]
Kicking users from channels might fail, because channels have additional restrictions on kicking: users can't be kicked from the #general channel and they cannot be kicked from the last channel they are in.
type restriction_error = [ `Restricted_action ]
If an action was attempted that the user does not have permission to, this error is returned.
type leave_general_error = [ `Cant_leave_general ]
Leaving the #general channel is not supported by Slack, an attempt do do so will trigger this error.
type topic_error = [ `Too_long ]
Slacko.topic types might be rejected by the Slack API because they are too long and can't be set.
type message_error = [ `Cant_delete_message | `Message_not_found ]
The Slacko.message might not exist or be impossible to delete for other reasons.
type message_length_error = [ `Msg_too_long ]
Slacko.message types, like Slacko.topic types might be too long to post.
type rate_error = [ `Rate_limited ]
Doing too many API requests in a certain timespan might cause a rate limitation to be applied by Slack. This is the error that results in that case.
type message_update_error = [ `Cant_update_message | `Edit_window_closed | `Message_not_found ]
Updating a Slacko.message might fail because the message was not found, couldn't be updated for some reason or because the time in which a message can be edited has passed.
type file_error = [ `File_deleted | `File_not_found ]
Handling files can result in multiple problems: the file wasn't found in the first place or it might have been deleted in the maintime.
type unknown_type_error = [ `Unknown_type ]
This error shouldn't ever be returned but serves as a catch-all in case the Slack API returns a new, unknown error type that Slacko doesn't yet understand.
type already_archived_error = [ `Already_archived ]
When trying to archive something that was already archived before, this error is returned.
type not_in_group_error = [ `Not_in_group ]
Doing an action in a Slacko.group when not being part of the group can fail.
type leave_last_channel_error = [ `Cant_leave_last_channel ] 
type last_member_error = [ `Last_member ]
An error when the user is the last member and can't do what he planned to do because that would cause the Slacko.channel not to have members anymore.
type oauth_error = [ `Bad_client_secret | `Bad_redirect_uri | `Invalid_client_id | `Invalid_code ]
These errors might be returned when the exchange of oauth authorization for a token has failed.
type presence_error = [ `Invalid_presence ]
Setting an invalid presence information is not supported.
type user_visibility_error = [ `User_not_visible ]
User is not visible, so action cannot be performed on them.
type authed_result = [ `Account_inactive
       | `Invalid_auth
       | `Not_authed
       | `Success of Yojson.Basic.json
       | `Unknown_error ]
API calls which require authentication will always return (at least) these error types.
type topic_result = [ `Account_inactive
       | `Channel_not_found
       | `Invalid_auth
       | `Is_archived
       | `Not_authed
       | `Not_in_channel
       | `Success of Yojson.Basic.json
       | `Too_long
       | `Unknown_error ]
Setting topics or purposes will result either in a success or one of these errors. Convenience type composed of subtypes.
type history_result = [ `Account_inactive
       | `Channel_not_found
       | `Invalid_auth
       | `Invalid_ts_latest
       | `Invalid_ts_oldest
       | `Not_authed
       | `Success of Yojson.Basic.json
       | `Unknown_error ]
Return value of a history related request.
type timestamp = float
Timestamps are usually represented as floats in OCaml, following suit
type token
Tokens are of a special type, use token_of_string to turn a string into a token.
type topic
The topic type represents a topic or a purpose message.
type message
The message represents a message to be posted.
type channel
A channel type, can be either a channel name (starting with a #) or a channel id.
type user
An user, represented by either a user name or a user id.
type group
A group, a private subset of users chatting together.
type sort_criterion =
| Score
| Timestamp
What criterion to use in search.
type sort_direction =
| Ascending
| Descending
Search result can be ordered in ascending or descending order.
type presence =
| Active
| Away
Presence can either be active or away.

Type construction helper functions



To build the types required in the API calls, you can use these helper functions.
val token_of_string : string -> token
Converts a string into a token.
val message_of_string : string -> message
Build a message from a string.
val topic_of_string : string -> topic
Build a topic out of a string. Slacko.topic types are also used to set purposes.
val group_of_string : string -> group
Construct a group out of a given string.
val user_of_string : string -> user
Converts a string into a user. Can be either a username or an user id.
val channel_of_string : string -> channel
Converts a string int a channel. Can be either a channel name or a channel id.
